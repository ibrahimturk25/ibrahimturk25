## 🚀 About Me

I'm a **Flutter & .NET full-stack mobile developer** from Ankara, Turkey. I own products end-to-end — from architecture decisions and backend APIs to App Store submission and production monitoring.

My flagship project, **Volley App**, serves **30,000+ active users** on iOS and Android. I built and maintain the entire stack: Flutter frontend, .NET 8 backend, real-time infrastructure, and Azure cloud deployment.

I'm also deepening my **native iOS** expertise with Swift — because understanding the platform at a native level makes me a better mobile developer across the board.

- 🏐 **30,000+ monthly active users** on a solo-built production app
- ⚡ Real-time systems with **SignalR** — live match tracking & instant messaging
- ☁️ **Azure**-deployed production environment with Application Insights monitoring
- 📱 Shipping to both **App Store** and **Google Play**
- 🎯 Computer Engineering graduate — Ataturk University (2024)
